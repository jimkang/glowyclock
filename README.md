glowyclock
==================

A little web site that shows a rainbow glowy LCD clock, like the kind my wife used to have. Forked from [SVG LCD Clock](http://codepen.io/christianhanvey/pen/imKIb/), added color tweening.

Installation
------------

1. Clone the repo. 
2.

    npm install

3.
    make build

Open in a browser.
